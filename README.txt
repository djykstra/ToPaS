TOtal PArsing Sistem (ToPaS) merupakan program terbatas untuk kalangan sendiri, distribusi dan penggunaan berdasarkan kepercayaan.


Prosedur instalasi:
1. install database (mysql atau mariadb minimal versi 5.1.x, disarankan 5.5.x), pastikan anda tahu ip, port, user dan password.
2. install library
3. install aplikasi
4. pastikan terinstall ms excel
5. pastikan terinstall driver printer (mempunyai setting printer)
6. pastikan modem berfungsi dgn baik dan benar, anda tahu  port dan speed nya.


Modem yg telah di test dan berjalan dgn baik:
1. huawei E220, E153
2. wavecom 2c2, 2303, 2403, 2406
3. mobile phone samsung galaxy chat bt5330
4. mobile phone cross d6x
5. alcatel internet modem
6. speedup internet modem
7. sierra wireless 312U, 313U
Jika anda memerlukan suatu modem dan atau hp dapat digunakan, silahkan kirimkan 1 unit untuk pengembangan.


Versi yg tersedia:
- Free Agent
- Pool
- Pool Agent
- Light
