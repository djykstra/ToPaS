TOtal PArsing Sistem (ToPaS) merupakan program terbatas untuk kalangan sendiri, distribusi dan penggunaan berdasarkan kepercayaan.


Prosedur instalasi:
1. install database (mysql atau mariadb minimal versi 5.1.x), pastikan anda tahu ip, port, user dan password.
2. install library
3. install aplikasi
4. pastikan mempunyai ms excel
5. pastikan mempunyai setting printer
6. pastikan modem berfungsi dgn baik dan benar, anda tahu  port dan speed nya.


Modem yg telah di test dan berjalan dgn baik:
1. huawei E220
2. huawei E153
3. wavecom 2c2, 2303, 2403, 2406
4. samsung galaxy chat bt5330
5. cross d6x
6. alcatel internet modem
7. speedup internet modem
8. sierra wireless 321u
Jika anda memerlukan suatu modem dan atau hp dapat digunakan, silahkan kirimkan 1 unit untuk pengembangan.


Versi yg tersedia:
- Free Agent
- Pool
- Pool Agent
- Light
