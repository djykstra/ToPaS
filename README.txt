TOtal PArsing Sistem (ToPaS) merupakan program terbatas untuk kalangan sendiri, distribusi dan penggunaan berdasarkan kepercayaan.

Sebelum anda menjalankan instalasi aplikasi, pastikan system telah terinstall 
- library tg diperlukan (tersedia installernya)
- MySQL/MariaDB 5.1.x, 5.5.x, port 3306, root password: admin.
- modem berjalan baik, diset speed 115200 dan anda tahu portnya.


Versi yg tersedia adalah
- Free Agent
- Pool
- Pool Agent
- Light
